## การพัฒนา

Clone Repo นี้ก่อน

```bash
git clone git@github.com:bombnp/cutuball74-backend.git
cd cutuball74-backend
git submodule init
git submodule update
```

จากนั้นอ่านเอกสาร

- [Static Readme](static/README.md) สำหรับ upload Front-end
